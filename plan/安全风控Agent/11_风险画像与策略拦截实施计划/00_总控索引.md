## 依赖合同

```text
Gateway 与 agent-service 共享 Redis key 合同：
risk:policy:short-link:disable:{domain}:{shortUri}
risk:policy:short-link:rate-limit:{domain}:{shortUri}
risk:policy:short-link:time-window:{domain}:{shortUri}
risk:policy:ip:block:{ipHash}
risk:rate:{domain}:{shortUri}:{ipHash}

画像与 Agent 合同：
risk-profile 只产出脱敏画像与 Top 10 异常候选。
SecurityRisk Agent 只消费画像事实，不全量扫描短链。
risk-center 查询 API 不走 Agent、不走 Tool、不走 LLM。

策略执行合同：
LIMIT_RATE 可在 HIGH + riskScore>=80 + 至少两个强 reasonCode 时自动执行。
DISABLE_SHORT_LINK、BLOCK_IP、LIMIT_TIME_WINDOW 必须人工确认。
```

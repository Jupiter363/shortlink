# 安全风控 Agent 验收清单

## 1. 架构验收

- [x] `campaignanalysisagent` 与 `securityriskagent` 包结构分离。
- [x] 默认 chat 请求仍走智能投放与分析 Agent。
- [x] `agentType=security-risk` 请求走安全风控 Agent。
- [x] 两个 Graph 使用不同 graphName。
- [x] 两个 Graph checkpoint 可通过 graphName 区分。

## 2. 功能验收

- [x] 支持分组级风险诊断。
- [x] 支持单短链风险诊断。
- [x] 支持访问明细抽样分析。
- [x] 输出 riskScore。
- [x] 输出 riskLevel。
- [x] 输出 reasonCode。
- [x] 输出 evidence。
- [x] 输出 recommendedActions。
- [x] 高风险处置只进入 pendingActions。

## 3. 安全验收

- [x] 不自动封禁 IP。
- [x] 不自动删除短链。
- [x] 不自动冻结分组。
- [x] 不绕过 admin internal tool API。
- [x] 不把原始 IP/user 发送给 LLM。
- [x] 不把原始 IP/user 写入 checkpoint。
- [x] 不提交真实 API Key、internal token、数据库密码。

## 4. 测试验收

- [x] `DefaultAgentRunHarnessTest` 覆盖 agentType 路由。
- [x] `AgentChatControllerTest` 覆盖 agentType 入参。
- [x] `SecurityRiskCardFactoryTest` 覆盖主要风险规则。
- [x] `DefaultSecurityRiskGraphExecutorTest` 覆盖工具调用、脱敏和 checkpoint。
- [x] `mvn -pl agent-service test` 通过。

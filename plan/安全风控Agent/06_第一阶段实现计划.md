# Security Risk Agent Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `agent-service` 中新增与智能投放分析 Agent 隔离的安全风控 Agent，实现只读风险诊断、风险卡片和 agentType 路由。

**Architecture:** `AgentRunHarness` 根据 `agentType` 路由到 `campaignanalysisagent` 或 `securityriskagent`。两个业务 Agent 分包实现各自 Graph、prompt、卡片工厂和测试，复用 Tool Registry、LLM client、Business Gateway 和 MySQL checkpoint。

**Tech Stack:** Java 17, Spring Boot, Spring AI Alibaba Graph, DeepSeek client, JUnit 5, AssertJ.

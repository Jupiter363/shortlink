# 02B 风控动作提案与执行器 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 SecurityRisk Graph 基于结构化画像、风险信号、明确指令和已持久化事件生成真实 `AgentPendingAction`，同时定义但暂不注册风控写执行器。

**Architecture:** `RiskScoringNode` 保留结构化 `RiskSignal`，`RiskActionProposalNode` 在事件持久化后调用确定性 factory。02B 只开放提案，不启用生产 executor；02C 完成策略事务和 Outbox 后再注册执行器 Bean。

**Tech Stack:** Spring AI Alibaba Graph、Java records、AgentProperties、Jackson、JUnit 5、Mockito、现有 risk-profile/risk-center/tool 基础设施。

# 待确认动作与策略一致性总控索引

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 以可独立验证和提交的五个阶段完成通用动作、风控提案、策略一致性、Admin 入口和最终回归。

**Architecture:** 先建立 Harness 状态机，再生成动作；只有策略 Outbox 完成后才注册并暴露写执行路径。所有阶段保持数据库和 API 向后兼容。

**Tech Stack:** Java 17、Spring Boot、Spring JDBC、MySQL/H2、Redis、OpenFeign、JUnit 5、Mockito、Figma。

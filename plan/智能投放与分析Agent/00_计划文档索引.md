## 已确认补充

1. LLM 默认使用 DeepSeek V4 Flash，密钥通过 `DEEPSEEK_API_KEY` 或 `LLM_API_KEY` 注入。
2. 第一阶段先完成 API 联调，随后交付 `agent-service` 独立 Agent Console。
3. `agent-service` 作为顶级 Maven module 独立建设。
4. Graph checkpoint 第一版直接 MySQL 持久化。
5. 独立 Agent Console 不直接绕过 admin 权限；生产链路仍以 admin/Gateway 用户边界为准。
6. 当前 Agent 不引入 MCP；工具层以进程内 Java Tool Facade 为主，避免为少量内部能力增加远程工具协议复杂度。

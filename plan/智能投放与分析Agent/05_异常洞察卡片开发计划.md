# 异常洞察卡片 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `agent-service` 中基于现有统计工具结果生成确定性的 `performance_insight` 与 `traffic_anomaly` cards。

**Architecture:** 不新增业务远程 API，不改 `AgentRunResult` DTO。新增一个独立的 insight card factory，从 `get_group_stats` / `get_short_link_stats` 的成功工具结果中派生结构化卡片，并在 Graph 组装响应与 LLM prompt 时复用这些确定性洞察。

**Tech Stack:** Java 17, Spring Boot, Spring AI Alibaba Graph, JUnit 5, AssertJ, Maven.

### Task 2: Implement deterministic insight card factory

**Files:**
- Create: `agent-service/src/main/java/com/nageoffer/shortlink/agent/agent/graph/CampaignInsightCardFactory.java`
- Modify: `agent-service/src/main/java/com/nageoffer/shortlink/agent/agent/graph/DefaultCampaignAnalysisGraphExecutor.java`

- [ ] **Step 1: Implement minimal factory**

Generate derived cards only for successful `get_group_stats` and `get_short_link_stats` executions.

Rules:

```text
traffic_anomaly / high_repeat_visits: pv / uv >= 5 and pv >= 50
traffic_anomaly / low_uip_share: uip / pv <= 0.2 and pv >= 50
traffic_anomaly / top_ip_concentration: top1 IP / pv >= 0.3 or top3 IP / pv >= 0.5
performance_insight / daily_pv_spike: latest daily pv >= 2x previous up-to-3-day average and delta >= 20
performance_insight / daily_pv_drop: latest daily pv <= 0.5x previous up-to-3-day average and delta >= 20
performance_insight / hour_concentration: max hour share >= 0.4 or top3 hour share >= 0.7
performance_insight / profile_concentration: browser/os/device/network/locale top1 ratio >= 0.7 and cnt >= 20
```

Daily trend rules must sort `daily` rows by `date` ascending first. The latest day is the max date, not the last item returned by SQL or the tool facade.

Top IP evidence must use masked IP strings.

- [ ] **Step 2: Wire factory into Graph**

`DefaultCampaignAnalysisGraphExecutor` appends derived cards after existing tool cards and appends derived insight context to the LLM prompt.

- [ ] **Step 3: Run green tests**

Run:

```bash
mvn -pl agent-service test "-Dtest=DefaultCampaignAnalysisGraphExecutorTest"
```

Expected: all graph executor tests pass.
